## Key Features

### AI-Powered Teleprompter
State-of-the-art teleprompter with advanced AI features:

- **Speech-to-text transcription** using Deepgram Nova-3 (industry-leading accuracy)
- **Voice-tracked auto-scrolling** that matches speaker pace
- **Script deviation detection** highlights when speakers go off-script
- **Real-time transcription display** shows exactly what's being said
- **Chroma key support** for green screen integration
- **High-quality audio** capture with Opus codec at 48kHz

## Technical Stack

eventools.io is built on modern, reliable, and scalable technology:

### Frontend
- **React 18.3+** with **TypeScript 5.7+** (strict mode)
- **Vite 5.4+** for lightning-fast builds
- **Tailwind CSS 3.4+** with custom glassmorphism design system
- **Radix UI** for accessible components (WCAG 2.1 AA compliant)

### Backend & Infrastructure
- **Supabase** (PostgreSQL, Auth, Realtime, Storage, Edge Functions)
- **Row-Level Security (RLS)** for multi-tenant data isolation
- **Deepgram Nova-3** for speech-to-text AI
- **Netlify** for global CDN and deployment

### Performance
- **Sub-second page loads** with code splitting
- **Offline-first architecture** with automatic sync
- **Real-time updates** via WebSocket connections
- **Mobile-optimized** for tablets and phones on-site
